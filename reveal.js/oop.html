Classes

Objects are defined in classes. A class is a template from which objects are instantiated.

public class myExampleClass {
  // properties
  private String exampleProperty;        

    // constructor
    public myExampleClass(String exampleParameter) {
      // initalize properties
      exampleProperty = exampleParameter;
    }

    //methods
    public String getExampleProperty() {
      return exampleProperty;
    }
    // more methods
}

Properties

Properties are simply variables that store data which belongs to the object

A key idea in OOP is encapsulation, this means objects hide their own data.

Here the private keyword means this property cannot be changed or viewed by code outside this object.

public class myExampleClass {
  // properties
  private String exampleProperty;        
}

Constructor

Every object (well non-static ones anyway) must have a constructor. This is a special method which gets called when we create an instance of an object.

    The constructor must be:
  • The first method,
  • public scoped
  • have the same name as the class
public class myExampleClass {
    // constructor
    public myExampleClass(String exampleParameter) {
      // initalize properties
      exampleProperty = exampleParameter;
    }
}

Constructors always return a new instance of the object, code needed to set the object up goes in the constructor.

Methods

Objects can contain all kinds of methods depending on what they will be used for.

Two particularly common type of methods are setters & getters, used to interact with object properties.

public class myExampleClass {
    //methods
    public String getExampleProperty() {
      return exampleProperty;
    }
    // more methods
}

This method returns the String identified by exampleProperty.

Using objects

Create objects by calling the constructor using the new keyword.

myExampleClass myObject = new MyExampleClass("some private data");

Access puplic properties or methods using dot syntax.

String result = myObject.getExampleProperty();

When we are calling a method, we must have brackets, even if there are no parameters.
