Selecting HTML Tags

Last week we were working with anchor, elements (aka. links)

Anchors are created with the <a> tag

A css rule to apply to all anchor elements, looks like this

a {
  color: red;
}

The rule name is followed by a colon and the rule property ends with a semicolon

Linking style sheets

To apply our style rules to the page the browser must be able to find our .css file.

A special <link> element must contain the relative file path to our style sheet.

This <link> element lives inside the <head> section of our .html file.

<head>
  <link rel="stylesheet"
  href="../css/myStyle.css">
</head>

Styles and JavaScript

We can also inject style rules directly using JavaScript

To do this we need to target elements by id property

We can add an id to any element, but id's must be unique.

<a id="myA">

Now we can use the console to select it

var e;
e = document.getElementById("myA");
e.style.color="red";
