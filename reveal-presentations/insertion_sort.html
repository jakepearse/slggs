Insertion sort

Using a nested while loop, assuming a Zero based array.

1. A ← The array to be sorted ;
2. FOR i ← 1 to length(A) - 1 :
  2.1 j ← i ;
  2.2 WHILE j > 0 and A[j-1] > A[j] :
    2.2.1 temp ← A[j] ;
    2.2.2 A[j] ← A[j-1] ;
    2.2.3 A[j-1] ← temp ;
    2.2.4 j ← j - 1 ;
  2.3 END WHILE
3. END FOR
