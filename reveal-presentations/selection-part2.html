  Action 2 ;
ENDIF

If Statement

Extended selections

Clumsy selection!

IF (parcelWeight < 5 kilos)
  Add 'light' sticker ;
ENDIF
IF (5 kilos < parcelWeight < 10 kilos)
  Add 'medium' sticker ;
ENDIF
IF (parcelWeight > 10 kilos)
  Add 'heavy' sticker ;
ENDIF

If Statement

Extended selections

Better selection

IF (parcelWeight < 5 kilos)
  Add 'light' sticker ;
ELSEIF (parcelWeight < 10 kilos)
  Add 'medium' sticker ;
ELSE
  Add 'heavy' sticker ;
ENDIF

Switch ... Case

Multi-way selection

SWITCH on Grade
Case: >=100
    Report “Perfect Score”
Case: > 89
    Report “Grade = A” ;
Case: > 79
    Report “Grade = B” ;
  ...
Default:
  Report “Grade = F” ;
End Case
