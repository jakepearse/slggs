Homework Review

File paths

Although many files might have the same file-name, you cannot have files with the same name in a single location

Because of this all files have a unique location - this is known as the file path

C:/Users/MrP/toDo.txt

Is distinct from

C:/Users/MrP/important/toDo.txt

Relative file paths

We don't always need to provide the complete (absolute) file path, to point to a file's location.

Often it's enough to provide the path from where in the file system we are right now.

This called the relative file path.

Hidden links

To enable relative file paths every directory has a hidden link to the directory above it

..

A relative file path might look like this

../images/myCatPicture.jpg

It means: from here go up one folder, then go into the images folder.

Hypertext links

We are familiar with hyper text links

http://www.silkom.co.uk/profile.jpg

The http:// part tells us that this is not a local file.

We use relative file paths to make links between pages that are all part of the same site

Hypertext links 2

In our html files Hyper text links are written like

Click Ye Here

There a 3 pieces to this anchor

  1. Opening tag (has the path inside)
  2. Link text (the text we will show)
  3. closing tag
