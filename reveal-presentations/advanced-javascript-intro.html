Input / Output

Input: Form elements or via prompt.

prompt("enter something");

Output: Modify DOM

console.log("foo");
document.writeln("bar");
alert("baz");

JavaScript console

Modern Browsers have a built-in JavaScript testing environment

Press F12 in Chrome to open the developer's panel

Look for the console tab and the > prompt

The console is a very useful way to test out your JavaScript code instantly.

Not a number

If we try to form an expression using a string and a number, we might get the special value

NaN

This is a particular oddity of JavaScript, although it's not a number it partially behaves like one!

For the most part, if you see NaN cropping up, it means you have tried to perform a mathematical operation on a string by mistake.
