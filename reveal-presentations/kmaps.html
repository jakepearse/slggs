### Properties of logical connectives #####Associative Conjunction *P ∧* (Q ∧ R)* = *Q ∧* (P ∧ R)* as in addition *a + (b + c)* = *(a + b) + c* --- #####Commutative Conjunction *P ∧ Q* = *Q ∧ P*
### Properties of logical connectives #####Distributive Disjunction *P ∧ (Q ∨ R)* = *(P ∧ Q) ∨ (P ∧ R)* We can reformulate disjunction, as in elementary algebra. Thanks De Morgan.
### Simplification True ∨ P = True True ∧ P = P ##### Proof that: *P ∨ P ∧ Q* = P ∧ has higher precedence than ∨ - P ∨ P ∧ Q = P ∨ P ∧ P ∨ Q - = P ∧ (True ∨ Q) - = P ∧ True - = P

### Karnaugh Maps K Maps provide a graphical way of minimising logical expressions *¬P∨¬Q* P\Q | 0 | 1 :--- | :---: | :--: **0** |1 |1 **1** | 1| 0
### Karnaugh Maps Blocks: - Must have no Zeros in them - Not be diagonal - Be as large as possible - Must encompass every 1 - Can overlap and "wrap around" - Be as few as possible
