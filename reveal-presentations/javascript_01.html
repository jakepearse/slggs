Programming in JavaScript - fundamentals

Programs are built up from expressions.

The simplest possible expression is a single value on its own this is called a literal.

197;

The semi-colon at the end tells JavaScript that this is the end of a statement.

Not a very useful expression, lets throw a few literals together using operators. 
197 - 7 + 10 / 2 * 10;

The value which the expression evaluates to is returned

Strings

"Hello World!";

Is another type of literal. Strings can be words, sentences or even a single character

When we join multiple strings together to make a new string, this is called concatenating them.

"Hello" + " " + "World" + "!";

What do we think will be returned if we make an expression like this?

"Hello" + 2 + "U";

Variables

In order to make our program do useful things we're going to need to stroe the values of our expressions.

We can use variables to 'grasp' our values

var name;

Now we have a variable labelled name we havn't given it a value so it has the special value undefined

name = "Joelle Rollo-Koster";

The single equals operator is used for assignment

The value returned by the expression on the right is assigned to the variable on the left
