###Functions - Map input to output. - Input is parameters or arguments (synomonus). - Output is a returned value. - For a given input, always return the same output. Pure functions are self-contained, they do not cause 'side effects' This type of function is often called a black-box function.
###Procedures Functions which are called only for their side effects are called procedures Any value returned from a procedure is usally ignored. ###Methods Functions (including procedures) which are contained within an *object* are called methods.

Function calls

The act of invoking a function is known as a function call, fuction calls are always followed by 0 or more paremeters enclosed in brackets

example function calls

f()
f(p1,p2,...,pn)

Not a function call

f
